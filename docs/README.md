# Screenshots & demo assets

Drop the images referenced by the main [README](../README.md) here. Suggested captures
(run `npm run dev`, open http://localhost:3000):

| Filename | What to capture |
| --- | --- |
| `demo.gif` | Short screen-recording: fill the form, generate a resume, export. Used as the hero. |
| `screenshot-intake.png` | The intake form on the left with document-type tabs and Generate button. |
| `screenshot-ats.png` | The green/amber/red ATS match score badge with missing-keyword chips. |
| `screenshot-resume.png` | A generated resume rendered in the preview panel. |
| `screenshot-export.png` | The Export PDF / Export DOCX / Copy buttons (and/or an exported file). |

Tips:
- Keep PNGs under ~500 KB so the README loads fast (resize to ~1400px wide).
- For the GIF, tools like ScreenToGif (Windows) or Kap (macOS) work well; aim for under ~5 MB.
- Use realistic but non-sensitive sample data in screenshots.
